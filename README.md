# 🌀 토션스프링 시뮬레이터 (Torsion Spring Simulator)

카트리지 도어 복귀용 **토션스프링**을 선정하기 전에, 스프링 스펙을 넣고
"이 스프링이 도어를 잘 밀어내는지" 미리 실험해 보는 단일 HTML 도구입니다.
초등학생도 알아볼 수 있게 쉬운 말로 만들었고, 어려운 공학 숫자는 접어두었습니다.

👉 **[index.html](index.html)** 파일을 브라우저로 열면 바로 동작합니다 (설치·서버 불필요).

## 무엇을 하나요?

1. **재료 선택** — 피아노선·크롬실리콘·스테인리스 등 (선경에 맞춰 강도 자동 계산)
2. **스프링 모양** — 철사 굵기·동그라미 크기·감김 수
3. **사용 조건** — 스프링 팔 길이·비틀림 각도
4. **카트리지 도어 크기·무게** — 가로·세로·높이(mm) + 무게(g)

그러면 결과로:

- 💙 내 스프링이 미는 힘 (N)
- 🛏️ **눕혀서 쓸 때**(도어 무게를 들어올림, 중력)와 🧍 **세워서 쓸 때**(마찰만)
  각각 필요한 힘을 막대로 비교하고 ✅/❌ 판정
- 🛡️ 스프링이 안 부러지는지(안전율) 체크
- 토크–각도 그래프 + 스프링 그림

## 계산 근거

표준 토션스프링 공식(SI: N·mm·MPa) 기반:

| 항목 | 식 |
|------|-----|
| 스프링 지수 | `C = D / d` |
| 각 강성(회전당) | `k = E·d⁴ / (10.8·D·n)` |
| 작동 토크 | `M = (k/360)·θ` |
| 굽힘응력(Wahl 보정) | `σ = Kᵢ·32M / (π·d³)` |
| 인장강도(선경 의존) | `Sut = A·d⁻ᵐ` |

도어 필요 힘 가정:
- 무게중심 거리 = 가장 긴 치수 ÷ 2
- 눕혀서(중력): 필요 토크 = 무게힘 × 거리
- 세워서(마찰만): 위 값의 30% (대략치)

> ⚠️ 선정 전 스크리닝용 근사 도구입니다. 피로 수명·영구변형(셋)·코일 간섭은
> 다루지 않으며, 정밀 설계 시 공급사 데이터시트로 보정하세요.

자세한 설계 배경은 [docs/design.md](docs/design.md) 참고.

## 검증

페이지 하단 "🔧 어른용 자세한 숫자 → 자가 테스트"에서, 손계산으로 검산한
기준 케이스(음악선 d=1.0, D=10, n=5, θ=90°)와 결과를 1% 오차 내로 비교합니다.
