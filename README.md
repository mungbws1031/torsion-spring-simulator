# 🌀 토션스프링 시뮬레이터 (Torsion Spring Simulator)

카트리지 도어 복귀용 **토션스프링**을 선정하기 전에, 스프링 스펙을 넣고
"이 스프링이 도어를 잘 밀어내는지" 미리 실험해 보는 단일 HTML 도구입니다.
초등학생도 알아볼 수 있게 쉬운 말로 만들었고, 어려운 공학 숫자는 접어두었습니다.

👉 **[index.html](index.html)** 파일을 브라우저로 열면 바로 동작합니다 (설치·서버 불필요).

## 무엇을 하나요?

1. **재료 선택** — 피아노선·크롬실리콘·스테인리스 등 (선경에 맞춰 강도 자동 계산)
2. **스프링 모양** — 철사 굵기·동그라미 크기·감김 수
3. **사용 조건** — 스프링 팔 길이·비틀림 각도
4. **카트리지 도어 크기·무게** — 가로·세로·높이(mm) + 무게(g)

그러면 결과로:

- 💙 내 스프링이 미는 힘 (N)
- 🛏️ **눕혀서 쓸 때**(도어 무게를 들어올림, 중력)와 🧍 **세워서 쓸 때**(마찰만)
  각각 필요한 힘을 막대로 비교하고 ✅/❌ 판정
- 🛡️ 스프링이 안 부러지는지(안전율) 체크
- 토크–각도 그래프 + 스프링 그림

## 계산 근거

표준 토션스프링 공식(SI: N·mm·MPa) 기반:

| 항목 | 식 |
|------|-----|
| 스프링 지수 | `C = D / d` |
| 각 강성(회전당) | `k = E·d⁴ / (10.8·D·n)` |
| 작동 토크 | `M = (k/360)·θ` |
| 굽힘응력(Wahl 보정) | `σ = Kᵢ·32M / (π·d³)` |
| 인장강도(선경 의존) | `Sut = A·d⁻ᵐ` |
| 정적 안전율 | `SF = (allowFrac·Sut) / σ` |

**피로(반복 내구) — Shigley 방법:**
- 스프링은 닫힘각(θ₀, 프리로드)~열림각(θ)을 왕복 → 응력진폭 `σa`, 평균응력 `σm`
- R=0 굽힘 피로강도 `Sr = frac·Sut` (작동 빈도·쇼트피닝별), 완전반전 환산 `Se = (Sr/2)/(1−Sr/2Sut)`
- 피로 안전율(Goodman) `nf = 1 / (σa/Se + σm/Sut)`

**프리로드:** 닫힘 위치 유지 토크 `= (k/360)·θ₀`.

**다리(레그):** 힌지 스프링은 양쪽 다리로 힘을 전달.
- 끝힘 `F = M / L` ↔ 원하는 힘으로 추천 다리 길이 역산 `L = M / F`
- 곧은 다리 뿌리 굽힘응력 `σ_leg = 32M/(πd³)` (곡률보정 없음) → 다리 안전율

도어 필요 힘 가정:
- 무게중심 거리 = 가장 긴 치수 ÷ 2
- 눕혀서(중력): 필요 토크 = 무게힘 × 거리
- 세워서(마찰만): 위 값의 30% (대략치)

> ⚠️ 선정 전 스크리닝용 근사 도구입니다. 피로·프리로드·정적 안전율은 1차 근사로
> 제공하지만, 영구변형(셋)·코일 간섭·다리 굽힘·전 구간(각도별) 토크 마진·실제 도어
> 형상은 다루지 않습니다. 부품 발주 전 공급사 데이터시트 + 정밀 피로 해석으로
> 반드시 최종 검증하세요. (피로 계수는 A228·302SS 기준 근사값)

자세한 설계 배경은 [docs/design.md](docs/design.md) 참고.
