# 토션스프링 시뮬레이터 설계 문서

작성일: 2026-06-19

## 1. 목적

토션스프링을 선정하기 전에, 스프링 스펙(재질·선경·코일지름·감김수·다리길이·작동각도)을
입력하면 **그 스프링이 낼 수 있는 토크/힘**을 계산하고, **버튼/힌지 복귀에 필요한 목표 힘**과
비교해 충분/부족을 판정하는 단일 HTML 시뮬레이터.

적용처: 버튼/힌지 복귀 스프링 (눌린 뒤 원위치로 돌아오게 하는 용도).

## 2. 형태

- 단일 파일 `torsion-spring-simulator.html` (이 repo의 `idea-sketch-maker.html`과 동일 방식).
- 외부 의존성 없음, 브라우저로 열면 즉시 동작.
- 좌측 입력 패널 / 우측 결과 + SVG 스프링 다이어그램 + 토크-각도 그래프.
- 입력값 변경 시 실시간 재계산.

## 3. 입력값

### 재질 (프리셋 + 직접입력)
선택 시 탄성계수 E와 인장강도 계수(A, m), 허용굽힘응력 비율이 자동 입력된다.
- 음악선(피아노선, ASTM A228)
- 오일템퍼선 (A229)
- 경강선 (A227)
- STS304 / STS302 (A313)
- 크롬실리콘 (A401)
- 크롬바나듐 (A232)
- 직접 입력 (E만 직접, 응력 체크는 인장강도 입력 시에만)

### 스프링 형상
- 선경 d (mm) — 필수
- 코일 평균지름 D (mm) — 필수. 외경(OD)/내경(ID) 입력 후 평균지름 환산 토글 제공
  (D = OD − d, D = ID + d)
- 활동 감김수 n (turns) — 필수

### 적용 (버튼/힌지 복귀)
- 다리(arm) 길이 = 모멘트암 L (mm)
- 작동 각도 θ (도)

### 목표값 (검증용)
- 필요한 복귀 힘 F_req (N) 또는 필요 토크 M_req (N·mm) — 한쪽 입력 시 자동 환산
  (모멘트암 기준 F_req = M_req / L)

## 4. 계산식 (표준 토션스프링 공식, SI: N, mm, MPa)

- 스프링 지수: `C = D / d`
- 각 강성 (회전당): `k_turn = E·d⁴ / (10.8·D·n)`  [N·mm/회전]
  - 10.8은 곡률·코일 마찰을 반영한 경험계수 (이론값 ≈10.2)
- 도당 강성: `k_deg = k_turn / 360`  [N·mm/도]
- 작동각도에서의 토크: `M = k_deg · θ`  [N·mm]
- 다리 끝 힘: `F = M / L`  [N]
- 곡률보정(Wahl 내측섬유): `K_i = (4C² − C − 1) / (4C(C − 1))`
- 굽힘응력: `σ = K_i · 32·M / (π·d³)`  [MPa]
- 인장강도(선경 의존): `Sut = A · d^(−m)`  [MPa]
- 허용굽힘응력: `σ_allow = allowFrac · Sut`  (정적, 재질별 0.80~0.87)
- 응력 안전율: `SF = σ_allow / σ`

## 5. 출력

- 스프링 강성 (N·mm/도), 스프링 지수 C
- 작동각도에서의 토크 M (N·mm) 및 다리 끝 힘 F (N)
- **복귀력 판정**: F(또는 M) vs 목표 → 충분/부족 + 여유율(%)
- **응력 안전 체크**: σ vs σ_allow → 안전/항복위험 + 안전율 SF
- **토크-각도 그래프**: 0~작동각도 선형 곡선 + 목표 토크 기준선 (SVG)
- **스프링 다이어그램**: 코일 + 양 다리 + 작동각도 시각화 (SVG)

## 6. 검증

- 계산은 순수 JS 함수(`computeSpring`)로 분리.
- 페이지 하단 "자가 테스트" 패널: 손계산으로 검증한 기준 케이스의 기대값과
  비교해 ✅/❌ 표시.
  - 기준 케이스: 음악선 d=1.0, D=10, n=5, E=207000, θ=90°, L=20
    - k_turn = 207000/(10.8·10·5) = 383.33 N·mm/회전
    - k_deg = 1.0648 N·mm/도
    - M(90°) = 95.83 N·mm
    - F = 4.79 N
    - C=10, K_i=1.0806, σ ≈ 1054 MPa
    - Sut(d=1) = 2211 MPa, σ_allow=1769 MPa, SF ≈ 1.68
  - 각 결과를 허용오차 1% 내로 검산.

## 7. 비포함 (YAGNI)

- 동적/피로 수명 계산, 영구변형(셋) 보정, 코일 간섭(body length) 검증은 제외.
- inch 단위계는 우선 제외 (SI만). 필요 시 후속 추가.
